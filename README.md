##  Acknowledgements

This repository contains the implementation of OGDC+, a few-shot relation extraction model developed as part of my doctor's dissertation. This model is based on the prototype network obtained by integrating the IRRI method, and uses the OGDC method to implement a distribution estimation strategy based on query instances, thereby generating more discriminative class prototypes.
